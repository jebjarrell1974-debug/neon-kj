# NEON KJ

  A karaoke show management system. Singers connect via WiFi, scan a QR code, search songs, and join the queue. The KJ manages rotation and controls a dual-display setup.

  ## Routes
  - `/` or `/host` — KJ control panel
  - `/singer` — Singer view (mobile, QR scan)
  - `/crowd` — Crowd-facing display (portrait TV)

  Built with React + Vite + Express + WebSockets + SQLite (libsql/drizzle).
